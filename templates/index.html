Cadastrar Imagens

Na seção de cadastro, as imagens serão geradas de maneira aleatória por meio de uma API integrada. Os usuários poderão visualizar novas imagens aleatórias ou escolher salvar uma delas no banco de dados. Ao selecionar uma imagem que desejam guardar, basta clicar em "Salvar" e adicionar um título personalizado. As imagens, junto com seus respectivos títulos, serão armazenadas no banco de dados local para futuras consultas.

Galeria

A galeria exibirá todas as imagens salvas pelos usuários, organizadas em um layout de grade. Isso permitirá que os usuários naveguem visualmente por suas coleções de forma fácil e intuitiva.
